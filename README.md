# TrueMimicVoice

Eliminates voice distortion effects for Mimics while preserving radio/transmitter voice effects.

## Features

- Removes Mimic voice distortion
- Preserves radio/transmitter voice effects
- No configuration required
- Tested on the latest version of MIMESIS

## Requirements

- MIMESIS
- MelonLoader 0.7.3+

## Installation

1. Install MelonLoader.
2. Copy `TrueMimicVoice.dll` into:

```
MIMESIS/Mods
```

3. Launch the game.

## Configuration

No configuration required. Simply install the mod and launch the game.
